# AI Engineer Roadmap / AI 工程师学习路线图

## Phase 1: Foundations (Weeks 1-4) / 阶段一：基础夯实（1–4周）
- Python (fastai course, CS61A) / Python 编程基础
- Git/GitHub / Git & GitHub 版本控制
- Basic Linux CLI / Linux 基础命令行
- Basic SQL / 基础 SQL 数据库操作

## Phase 2: ML Fundamentals (Weeks 5-8) / 阶段二：机器学习基础（5–8周）
- Supervised/Unsupervised Learning / 监督学习与无监督学习
- Metrics: Accuracy, Precision, Recall, F1, AUC / 评估指标：准确率、精确率、召回率、F1、AUC
- Train/Val/Test splits, overfitting, regularization / 数据集划分、过拟合、正则化
- Pandas, NumPy, Matplotlib / 数据处理与可视化库
- Kaggle beginner competitions / Kaggle 入门竞赛

## Phase 3: Deep Learning (Weeks 9-12) / 阶段三：深度学习（9–12周）
- Neural network basics / 神经网络基础
- CNNs, RNNs, Transformers / CNN、RNN、Transformer
- PyTorch or TensorFlow / PyTorch 或 TensorFlow 框架
- Hugging Face Transformers, Diffusers / HF 相关库使用
- Fine-tune pre-trained models / 微调预训练模型

## Phase 4: LLM & AI Engineering (Weeks 13-16) / 阶段四：大模型 & AI 工程（13–16周）
- LLM architecture & scaling laws / LLM 架构与缩放规律
- Prompt engineering / 提示词工程
- RAG (Retrieval-Augmented Generation) / RAG 检索增强生成
- Embeddings & vector databases / 向量嵌入与向量数据库
- Fine-tuning small/large models / 大小模型微调
- Function calling, tool use / 函数调用与工具使用
- Evaluation & hallucination mitigation / 模型评估与幻觉缓解

## Phase 5: Production & MLOps (Weeks 17-20) / 阶段五：生产部署 & MLOps（17–20周）
- API design (FastAPI) / API 开发（FastAPI）
- Docker & containers / Docker 容器化
- Cloud basics (AWS/GCP/Azure) / 云服务基础
- Model deployment / 模型部署
- Monitoring, logging, alerting / 监控、日志、告警
- CI/CD for ML / ML 持续集成部署
- Experiment tracking (Weights & Biases) / 实验跟踪

## Phase 6: Build Portfolio (Ongoing) / 阶段六：打造作品集（持续进行）
- 3+ end-to-end projects: / 3 个以上端到端项目：
  - RAG chatbot / RAG 智能问答机器人
  - Fine-tuned classifier / 微调分类模型
  - AI agent with tools / 带工具能力的 AI 智能体
- Document on GitHub + blog post / GitHub 文档 + 博客总结

## Daily Habits / 日常习惯
- Code 1-2 hours / 每天编码 1–2 小时
- Read 1 paper / 每天读 1 篇论文
- Build in public / 公开构建、持续分享
